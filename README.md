# Sistema de Estoque em Python

Projeto desktop feito com Tkinter para gerenciamento de produtos.

## Funcionalidades
- Adicionar produtos
- Remover produtos
- Salvar em JSON
- Gerar planilha Excel

## Tecnologias
- Python
- Tkinter
- JSON
- OpenPyXL

## Como rodar
pip install -r requirements.txt
python main.py

## Download

Baixe o executável:
- sistema-estoque.exe

## Como usar
- Abra o .exe
- Adicione produtos
- Gere planilha automaticamente
