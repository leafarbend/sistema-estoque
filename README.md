# Sistema de Estoque em Python

Projeto desktop feito com Tkinter para gerenciamento de produtos.

## Funcionalidades
- Adicionar produtos
- Remover produtos
- Salvar em JSON
- Gerar planilha Excel

## Tecnologias
- Python
- Tkinter
- JSON
- OpenPyXL

## Download

Baixe o executável:
- sistema-estoque.exe

- ## 🚀 Como usar

### Opção 1 (mais fácil)
1. Baixe o arquivo sistema-estoque.exe
2. Execute o arquivo
3. Comece a usar

### Opção 2 (código)
pip install -r requirements.txt
python main.py
